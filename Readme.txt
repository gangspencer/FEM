FEM (FILE ENCRYPTION MODULE)
Created by Spencer Gang

FEM (File Encryption Module) is a program designed to intake a text file,
read char by char, encrypt and save as a new file.

Current method of encryption:
  -User inputs path for both file to be encrypted and encrypted file
  -User inputs a key
  -Text file is read char by char adding key to each char and changing its ASCII value.
  -Char is written to final file
